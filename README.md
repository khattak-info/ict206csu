# ict206csu

Repository structure for ITC206 - Programming in Java 1 teaching materials.

## Sections
- `section-01/`
- `section-02/`
- `section-03/`

Each section contains 11 topic folders (`topic-01` to `topic-11`).
Every topic folder includes:
- `README.md` with rich teaching notes/support content
- `src/` for in-class example code (kept intentionally empty except `.gitkeep` to preserve folder structure in Git)
