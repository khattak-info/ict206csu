# Topic 02: Elementary Programming

## Focus
Students learn Java syntax fundamentals: variables, data types, expressions, console input/output, and simple arithmetic problem solving.

## Learning outcomes
Students should be able to:
- declare and initialize variables using appropriate types;
- use arithmetic operators and precedence rules;
- perform type conversion (cast/widening) safely;
- read values from console input and display formatted output.

## Key concepts
- primitive types (`int`, `double`, `char`, `boolean`)
- assignment and expressions
- constants (`final`)
- `Scanner` for input
- basic numeric algorithms (area, average, unit conversion)

## In-class sequence
1. Review program structure and imports.
2. Build simple calculator-style examples.
3. Add input validation discussion (where relevant at beginner level).
4. Compare integer vs floating-point results.

## Practice ideas
- temperature converter (C/F)
- BMI calculator
- simple pay calculator with hours and rate

## Common mistakes to address
- integer division surprises;
- missing `nextLine()` handling after numeric input;
- uninitialized variables and scope confusion.

## Assessment connection
Supports foundational algorithmic thinking needed for selection and loops.

## Folder usage
Use the `src` directory for all elementary programming examples.
