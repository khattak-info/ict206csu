# Topic 11: Abstract Classes and Interfaces

## Focus
Students finalize the OO pathway with abstraction mechanisms that support flexible and extensible designs.

## Learning outcomes
Students should be able to:
- define and use abstract classes;
- implement interfaces and contract-based design;
- compare abstract classes and interfaces for appropriate use;
- build small polymorphic systems using shared contracts.

## Core content
- abstract methods/classes
- interface declaration and implementation
- multiple-interface implementation
- programming to an interface

## In-class activities
- create an abstract base with concrete subclasses;
- implement a shared interface across unrelated classes;
- run polymorphic processing using interface references.

## Revision alignment
This topic supports final revision of:
- encapsulation
- inheritance
- polymorphism
- design quality and maintainability

## Folder usage
Use the `src` directory for abstraction/interface examples and revision exercises.
