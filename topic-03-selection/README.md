# Topic 03: Selection

## Focus
Students apply decision-making in programs using boolean expressions, relational operators, and branching statements.

## Learning outcomes
Students should be able to:
- write correct boolean conditions;
- choose between `if`, `if-else`, `if-else-if`, and `switch`;
- design and test multi-branch logic;
- avoid unreachable or contradictory conditions.

## Core content
- relational and logical operators
- nested selections
- guard conditions
- `switch` expressions/statements (as appropriate to level)

## In-class activities
- classify numbers (positive/zero/negative);
- grade calculator with bands;
- menu-driven examples using `switch`.

## Algorithm testing emphasis
Encourage table-driven testing:
- normal case
- boundary case
- invalid input case

## Common pitfalls
- using `=` instead of `==` in conditions;
- impossible conditions due to ordering;
- forgetting `break` in classic `switch`.

## Folder usage
Use the `src` directory for all selection examples and test cases.
