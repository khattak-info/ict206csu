# Topic 06: Arrays

## Focus
Students store and process collections of values using single-dimensional arrays.

## Learning outcomes
Students should be able to:
- declare, create, and initialize arrays;
- iterate through arrays with indexed loops and enhanced `for`;
- compute statistics from array data;
- search and update array contents.

## Core content
- array declaration and memory model
- index-based access
- bounds checking
- basic algorithms: linear search, min/max, average

## In-class activities
- student marks processing;
- frequency counting basics;
- array input/output with loops.

## Common pitfalls
- `ArrayIndexOutOfBoundsException` from bad loop limits;
- confusing array length with last index;
- forgetting to initialize before access.

## Extension discussion
Brief intro to parallel arrays and when to transition to objects.

## Folder usage
Use the `src` directory for all array examples.
