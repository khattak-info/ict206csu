# Topic 04: Loops

## Focus
Students develop repetition logic using `while`, `do-while`, and `for` loops, including accumulators and counters.

## Learning outcomes
Students should be able to:
- choose an appropriate loop type for a problem;
- trace loop execution and termination;
- build count-controlled and sentinel-controlled loops;
- compute aggregate results (sum, min/max, averages).

## Core patterns
- counting loops
- sentinel loops
- running total and count
- nested loops for simple patterns/tables

## In-class activities
- multiplication table generator;
- repeated input until sentinel value;
- factorial and power calculations.

## Testing and tracing
- dry-run with trace tables;
- check initial condition, update step, stop condition;
- identify and prevent infinite loops.

## Common pitfalls
- off-by-one errors;
- failing to update loop variable;
- incorrect loop condition boundaries.

## Folder usage
Use the `src` directory for all loop-driven examples.
