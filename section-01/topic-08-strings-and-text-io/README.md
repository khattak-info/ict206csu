# Topic 08: Strings and Text I/O

## Focus
Students work with textual data using `String` operations and file input/output with text files.

## Learning outcomes
Students should be able to:
- manipulate strings using common methods;
- parse and validate simple textual input;
- read from and write to text files safely;
- handle exceptions for file operations at beginner level.

## Core content
- immutability of `String`
- substring/search/replace/trim/case methods
- basic formatting
- file reading/writing (`Scanner`, `PrintWriter`, or modern APIs)

## In-class activities
- word/line counters;
- simple CSV-like parsing;
- write processed output to a new file.

## Common pitfalls
- using `==` for string comparison instead of `.equals()`;
- not closing resources;
- fragile parsing with missing delimiters.

## Folder usage
Use the `src` directory for string and file I/O examples.
